About us

We started in early 2018 setting up meetups for the community to explore everything that is Infrastructure as code. After running many successful meetups on subjects like: How to Github, server build automation, visualising your infrastructure with Elastic Stack, Terraform in production, A Cloud Guru - teaching the world to cloud, and Running stateful applications in Kubernetes, we started to move into offering an Introduction to Devops tools training course. Our meetups continue as always, and over the last year, we've had the pleasure of working with the FT, Worldpay and the Bank of England to upskill some of their IT teams in many of the most popular Devops tools in-use today. We are constantly evolving our training and will soon be offering an introduction to containers, Kubernetes and prometheus (similar to Kubernetes - the hard way). Feel free to join our mailing list at the bottom of the page, join and attend our next meetup, and if you want to learn about Devops tools, reach out, and we would be happy to help.

About our training

Join us for a day of devops tools on Wednesday 11th November 2020.

Our DevOps for Operations course focus's on provisioning, managing, orchestrating, testing & monitoring of AWS cloud infrastructure. Start by building a custom AWS AMI image using Packer. Then use Terraform to build all your servers, Ansible to install and configure software, Gitlab/Github for source control, Jenkins to orchestrate the CI/CD pipeline with integrated testing using ServerSpec on git commit, and finally Elasticstack/Grafana to visualise server performance.

Our Kubernetes course focus's on building a basic container image and uploading to DockerHub, setting up a 2 node Kubernetes cluster and monitoring the cluster with Prometheus and Grafana. We go through how to set up and test autoscaling for our application and we also give an intro into installing and using Helm (the package manager for Kubernetes). We cover how to set up load-balancing in AWS for our application and link our application to a web domain so anyone can visit it on the internet. This course is completely hands-on. Each attendee will create their own container and Kubernetes cluster. Theyll run through all the setup themselves and get a good overview of how everything hangs together in a safe lab environment. They keep all the code I provide as well as the student workbooks so they can reference what they've learned once back home or in the office

    Our Course

    Join us for a full-day hands-on devops tools workshop focusing on provisioning, managing, orchestrating, testing & monitoring of AWS cloud infrastructure using these tools.

    Ansible

    Ansible is an open-source software provisioning, configuration management, and application-deployment tool. It runs on Linux and can configure both Linux and Windows systems

    Amazon Web Services

    Amazon Web Services is a subsidiary of Amazon that provides on-demand cloud computing platforms and APIs to individuals, companies, and governments, on a metered pay-as-you-go basis

    Terraform

    Terraform is an open-source infrastructure as code software tool created by HashiCorp. It enables users to define and provision a datacenter infrastructure using a high-level configuration language known as Hashicorp Configuration Language, or optionally JSON.

    Jenkins

    Jenkins is a free and open source automation server. Jenkins helps to automate the non-human part of the software development process, with continuous integration and facilitating technical aspects of continuous delivery.

    ELK Stack

    The ELK Stack is a collection of three open-source products. Elasticsearch, Logstash, and Kibana. They are all developed, managed, and maintained by Elastic.

    Gitlab

    GitLab is a web-based DevOps lifecycle tool that provides a Git-repository manager providing wiki, issue-tracking and CI/CD pipeline features, using an open-source license, developed by GitLab Inc

    Hashicorp Packer

    Packer is a tool for building images for cloud platforms, virtual machines, containers, and more from a single source configuration.

    ServerSpec

    ServerSpec is a Ruby tool that allows you to write simple tests to validate that a server has been correctly configured.

    My story

    Dennis McCarthy

    Founder

    The world is a different place now to when I started out in IT (back in 1996!). Meetups were non-existent, and training courses were all about one product and how to use it. Now we live in a world with endless possibilities and 1000s of different ways to do anything! This is why I created London Infrastucture as code. I want to show people how to successfully use some of the most popular devops tools, take this knowledge, and use it back in their job or a personal project. I've been to many meetups in my time and sometimes I'm left thinking, "that was great, but how do I do it!". My meetup will always give you a way in with starter code and lots of demos. Born out of my meetup talks, I created the "Introduction to devops tools" hands-on workshop. It will teach you how to build, setup, test and monitor a CI/CD workflow in AWS using some of the best tools on the market. It's an exciting time to be in Tech so come along to my meetup and join in!
